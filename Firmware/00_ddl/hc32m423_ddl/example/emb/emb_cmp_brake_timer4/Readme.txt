平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示EMB模块的电压比较器比较结果制动功能。

说明：
EMB检测电压比较器比较结果标志位，
1、比较器比较结果标志位置起时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调
   函数
2、EMB中断回调函数内，然后清除比较器比较结果标志位和EMB状态，PWM正常输出

使用步骤
================================================================================
1）测试板J6-RXD3(TIM4_OUH)、J6-CTSRTS3(TIM4_OUL)引脚与示波器连接，J6-P13与VCC连接
2）编译、下载并运行
3）观察LED灯，绿色灯点亮
4）观察示波器，示波器J6-RXD3(TIM4_OUH)、J6-CTSRTS3(TIM4_OUL)通道显示周期脉冲（1Hz）
5）J6-P13与GND连接
6）观察LED灯，绿色灯熄灭
7）观察示波器，示波器J6-RXD3(TIM4_OUH)、J6-CTSRTS3(TIM4_OUL)通道显示低电平
8）J6-P13与VCC连接，按下并释放SW2
9）重复步骤3）~步骤8）
