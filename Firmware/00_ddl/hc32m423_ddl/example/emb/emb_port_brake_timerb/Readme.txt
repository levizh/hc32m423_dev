平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示EMB模块的端口输入刹车功能。

说明：
EMB检测端口输入电平，
1、满足端口输入电平时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调函数
2、EMB中断回调函数内，等待按键释放，然后清除EMB状态，PWM正常输出

使用步骤
================================================================================
1）测试板J6-TXD3(TIMB_1_PWM1)、J6-I2C_SCL(TIMB_1_PWM2)引脚与示波器连接，J6-TXD4与J4-KEY相连
2）编译、下载并运行
3）观察示波器，示波器J6-TXD3、J6-I2C_SCL通道显示周期脉冲
4）按下SW2按键
5）观察示波器，J6-TXD3、J6-I2C_SCL通道电平为低
6）释放SW2按键
7）观察示波器，示波器J6-TXD3、J6-I2C_SCL通道显示周期脉冲
8）重复步骤3） ~步骤7）
