平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示INTC模块EXINT及NMI Pin的检测功能。

说明：
本样例通过开发板上的KEY1~KEY4，展示MCU外部中断、NMI Pin中断功能；
同时通过代码操作，演示了软件中断响应的功能。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序并运行；
2）首先会进入Software Ch14的中断入口，点亮LED_RGB;
3）按下KEY1~KEY4，观测LED状态变化
   KEY1（EXINT5，下降沿），按下后，LED_R交替亮灭；
   KEY2（EXINT4，上升沿），按下并抬起后，LED_G交替亮灭；
   KEY3（NMI，下降沿），按下后，LED_B交替亮灭；
   KEY4（EXINT7，双沿），按下或抬起后，LED_RGB同时交替亮灭；
