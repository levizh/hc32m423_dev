平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本例程展示了TIMER2异步时钟源的配置，实现了用异步时钟作为时钟源定时50ms。

说明：例程通过宏定义选择LRC（32.768KHz）或从CLKA（P30）引脚输入的时钟作为异步时钟源。
      如果选择从CLKA（P30）引脚输入的时钟作为异步时钟源，请输入频率为10KHz的时钟。

使用步骤
================================================================================
1）打开工程并重新编译，将程序下载至目标板；
2）如果时钟源选择的是LRC，会直接看到RGB LED的红色LED以10Hz的频率闪烁；
3）如果时钟源选择的是从CLKA（P62）输入的时钟，请从CLKA输入10KHz的时钟，会看到RGB LED的
   红色LED以10Hz的频率闪烁；

================================================================================
注意
================================================================================
1) 从TRIGA输入的异步时钟的最高频率为25MHz，PWM输出功能、捕获功能均可使用异步时钟；
2）在使用异步时钟时，修改计数值（CNTAR）、基准值（CAPAR）、启动位（BCONR.CSTA）、
   状态位（STFLR.CMFA）时，Timer2从接收到写动作后经过3个异步时钟周期才将修改值写入对应的
   寄存器中。
3）在使用异步时钟时，连续对计数值（CNTAR）、基准值（CAPAR）、启动位（BCONR.CSTA）、
   状态位（STFLR.CMFA）进行写操作时，需间隔至少3个异步时钟周期。
