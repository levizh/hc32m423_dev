平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示如何使用Timer4专用比较功能——延迟触发模式。

说明：
Timer4专用比较功能比较触发模式：计数器单元CNT与OCO比较匹配后，经过SEVT延迟后，产
生专用比较信号时间，该事件启动ADC采样。

================================================================================
使用步骤
================================================================================
1）测试板J3-P63引脚与示波器连接；
2）编译、下载并运行；
3）观察示波器，示波器显示周期脉冲（1Hz），周期时间与CNT周期一致。
4）测试板J7-P22引脚与GND相连
5）通过IDE watch窗口，查看m_au16AdcSbVal值为0x0000
6）测试板J7-P22引脚与VCC相连
7）通过IDE watch窗口，查看m_au16AdcSbVal值为0x0FFF
8）重复步骤3） ~步骤7）
