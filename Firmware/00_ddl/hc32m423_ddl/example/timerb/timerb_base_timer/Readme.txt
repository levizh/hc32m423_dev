平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示TimerB模块基本计数功能。

说明：
1、本样例设置TimerB为锯齿波、向上计数模式，使能计数上溢中断，中断处理程序翻转RGB
   绿色灯引脚，RGB绿色灯闪烁；
2、通过硬件启动TimerB(TIMB_1_PWM1边沿), 硬件清零TimerB(TIMB_1_PWM3）。

使用步骤
================================================================================
1）按照如下方式，连接引脚：
   J4-P01 <----> J6-P71
   J4-P02 <----> J6-P72
2）打开工程并重新编译；
3）启动IDE的下载、运行程序；
4）按下、释放K1按键，RGB绿色灯不停闪烁，闪烁频率为1Hz；
5）当RGB绿色灯为熄灭状态时，反复快速按下、释放K2按键，RGB绿色灯一直保持熄灭状态；
6）当RGB绿色灯为亮状态时，反复快速按下、释放K2按键，RGB绿色灯一直保持亮状态；
7）释放K2按键，RGB绿色灯不停闪烁，闪烁频率为1Hz；
8）重复步骤5） ~ 7）。
