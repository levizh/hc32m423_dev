平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示TimerB模块输入捕获功能。

说明：
1、本样例设置TimerB为锯齿波、向上计数模式，启动条件为外部中断事件
2、TimerB通道3使能输出比较功能，比较匹配和TimerB通道3使能输出比较功能，比较匹配和
周期匹配翻转输出极性，比较匹配TIMB_1_PWM3产生下降沿；
3、TimerB通道1和2使能输入捕获功能，TimerB通道1和2在TIMB_1_PWM3下降沿时，捕获输入。

================================================================================
使用步骤
================================================================================
1）将J6-P72(TIMB_1_PWM1)、J6-P75(TIMB_1_PWM2)、J6-P71(TIMB_1_PWM3)相连
2）打开工程并重新编译
3）启动IDE的下载和调试功能全速运行
4）按下、释放K1按键，RGB绿色LED点亮，表示已触发启动TimerB；
5）通过IDE的Live Watch功能，查看捕获输入缓存变量m_stcEvenUnitCmpBuf、
   m_stcOddUnitCmpBuf为TimerB通道3 TMRB_CMPAR值加1
