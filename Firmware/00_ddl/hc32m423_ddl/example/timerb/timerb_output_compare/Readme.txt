平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示TimerB模块输出比较功能。

说明：
本样例设置TimerB为三角波模式，TimerB计数周期值为500ms，通道1比较值通过寄存器缓存
功能，依次设置为周期值的[1/4, 1/2, 3/4]，产生PWM波形。

================================================================================
使用步骤
================================================================================
1）J6-P72(TIMB_1_PWM1)引脚与示波器相连
2）打开工程并重新编译；
3）启动IDE的下载程序功能，关闭IDE下载界面；
4）按下、释放K1按键，RGB绿色LED点亮，表示已触发启动TimerB；
5）观察示波器，J6-P72通道波形及高低电平时间如下。
       625ms-625ms-500ms-375ms-375ms-500ms-625ms-625ms-500ms-375ms-375ms-500ms
